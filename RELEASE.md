# 🚀 Release Notes: ScheTime v1.0.0

Kami dengan bangga merilis versi stabil pertama **ScheTime**! Aplikasi ini siap membantu produktivitasmu dengan sistem pengingat yang handal.

## 📥 Download APK

Kami menyediakan 3 varian APK berdasarkan arsitektur prosesor perangkatmu. Silakan pilih yang sesuai agar aplikasi berjalan optimal:

| File APK | Deskripsi Arsitektur | Target Perangkat |
| :--- | :--- | :--- |
| **`app-arm64-v8a-release.apk`** | **64-bit ARM** | **Rekomendasi Utama.** Cocok untuk hampir semua HP Android keluaran tahun 2016 ke atas (Samsung S-series, Xiaomi Note series, Pixel, dll). |
| **`app-armeabi-v7a-release.apk`** | **32-bit ARM** | Untuk HP Android model lama atau kelas *entry-level* yang masih menggunakan prosesor 32-bit. |
| **`app-x86_64-release.apk`** | **x86_64** | Digunakan khusus untuk **Emulator Android** (seperti di Android Studio/BlueStacks) atau beberapa tablet/laptop Chromebook. |

> **Tips:** Jika kamu tidak yakin, coba instal versi **`arm64-v8a`** terlebih dahulu. Jika muncul pesan "App not compatible", gunakan versi **`armeabi-v7a`**.

---

## 🆕 Apa yang Baru?

- **Penyempurnaan Sistem Reminder**: Fitur pengingat terpisah dari Todo List.
- **Sinkronisasi Kalender**: Otomatis masuk ke Google Calendar atau kalender bawaan HP.
- **Alarm Layar Penuh**: Suara nyaring dan tampilan alarm yang tidak bisa terlewatkan.
- **Custom Ringtone**: Kamu bisa memilih lagu MP3 sendiri untuk alarm.
- **Volume Maksimal**: Alarm otomatis berbunyi di volume 100% dengan efek *fade-in*.
- **Optimasi Waktu**: Standarisasi zona waktu Asia/Jakarta.

---

## 🛠️ Cara Instalasi

1. Download file APK yang sesuai di atas.
2. Jika muncul peringatan keamanan, aktifkan opsi **"Allow from this source"** atau **"Install unknown apps"** di pengaturan HP-mu.
3. Buka file APK dan klik **Install**.
4. Berikan semua izin (*permissions*) yang diminta (Kalender, Notifikasi, Alarm) saat pertama kali membuka aplikasi agar fitur berjalan lancar.
